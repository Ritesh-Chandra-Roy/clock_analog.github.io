# clock_analog.github.io
Analog clock developed using javascript concepts like querySelector(), setInterrval() and setDate() function. Used CSS transform and transition properties to graphical represent the clock's hands movement.

--BONUS--

Addaed featture to display Weather using OpenWeatherMap API.
Added Feature to get current location's weather using Geolocation API.
Added Feature to display Random Quotes related to time and life along with author name. 

#NOTE 
Mine API key won't work on your local machine. Please login to https://home.openweathermap.org/ and use your own API key. Just replace the value of APIkey variable inside clock.js file with your API key. Rest all will remain same.
